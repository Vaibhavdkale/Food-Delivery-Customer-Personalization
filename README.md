# 🍔 Food Delivery Customer Personalization

## 📌 Project Overview

This project analyzes how customer personalization features influence add-on buying behaviour in food delivery applications.

The study focuses on user behaviour in **Zomato** and **Swiggy** using survey data collected from **118 respondents**.

---

## 🎯 Objectives

- Study customer personalization in food delivery apps.
- Analyze factors affecting add-on purchases.
- Compare customer behaviour across platforms.
- Provide business recommendations.

---

## 🛠️ Tools Used

- Microsoft Excel
- Power BI
- Statistical Analysis
- Survey Research

---

## 📊 Dataset

- Survey Responses: 118
- Platform: Zomato & Swiggy
- Type: Primary Data

---

## 📈 Key Findings

- Personalized recommendations positively influenced add-on purchases.
- Discounts and combo offers significantly increased purchase decisions.
- High-income users showed the highest average order value.
- Customers preferred personalized recommendations over generic offers.
