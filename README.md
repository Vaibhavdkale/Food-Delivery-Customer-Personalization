# 🍔 Food Delivery Customer Personalization & Add-On Buying Behaviour

## 📌 Project Overview

This project analyzes how customer personalization influences add-on buying behaviour in food delivery applications using survey data collected from **118 respondents**. The study focuses on customer behaviour on **Zomato** and **Swiggy** and provides data-driven business recommendations.

---

## 🎯 Objectives

- Analyze customer personalization.
- Study add-on buying behaviour.
- Compare customer preferences across platforms.
- Provide business recommendations using analytics.

## 📈 Key Findings

- Personalized recommendations positively influenced add-on purchases.
- Discounts and combo offers had the highest impact on purchase decisions.
- High-income users recorded the highest Average Order Value (AOV).
- Customers preferred personalized recommendations over generic suggestions.

---

## 💡 Business Recommendations

- Improve personalized recommendation algorithms.
- Introduce customized combo offers.
- Strengthen loyalty and reward programs.
- Use customer behaviour analytics to improve retention.
